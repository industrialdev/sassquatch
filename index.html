Getting Started

This project boilerplate is intended to serve as a front end starting point for all of industrial's projects. This project provides a baseline for building responsive and accessible website structures.

Getting started with the project boilerplate is simple:

  • Copy the assets folder from this project into the project theme folder that you would like to apply the boilerplate to
  • Drop the gulpfile from this boilerplate into the root of your project
  • Ensure that all paths in the gulpfile are relative to your projects folder structure
  • Navigate to your project in the command line
  • Run the command: npm install to install dependencies
  • Run the command: gulp to compile your assets and watch for changes

The Grid

The industrial boilerplate uses a custom grid system. Using the default class definitions outlined below will result in a flexible grid that is not responsive. Breakpoint prefixes can be used if you want the columns to break to different widths at various screen sizes.

eg. .col_1, .col_sm-1, .col_md-1, .col_lg-1, col_xl-1
